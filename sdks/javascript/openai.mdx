# OpenAI Integration

The Foil SDK provides seamless integration with the OpenAI SDK, offering multiple ways to trace your API calls.

<Note>
  Building with the **OpenAI Agents SDK** (`@openai/agents`)? See the [OpenAI Agents SDK](/sdks/javascript/openai-agents) page for tracing multi-agent workflows, handoffs, and guardrails.
</Note>

<Tip>
  **Full examples**: [customer-support-agent](https://github.com/getfoil/foil-examples/tree/main/customer-support-agent) — same agent in two flavors: [SDK tracing](https://github.com/getfoil/foil-examples/blob/main/customer-support-agent/agent-native-sdk.js) and [auto-instrumentation](https://github.com/getfoil/foil-examples/blob/main/customer-support-agent/agent-otel.js)
</Tip>

## Quick Start (Recommended)

Use `ctx.llmCall()` for full tracing with nested span trees — works with any model and gives you tool spans under LLM calls:

```javascript
const { Foil } = require('@getfoil/foil-js');
const OpenAI = require('openai');

const openai = new OpenAI();
const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
});

await foil.trace(async (ctx) => {
  const response = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: 'Hello!' }],
    });
  });

  console.log(response.choices[0].message.content);
}, { name: 'chat' });
```

## Tool/Function Calls with executeTools

The `ctx.executeTools()` helper automatically executes and traces all tool calls from an OpenAI response:

```javascript
await foil.trace(async (ctx) => {
  const messages = [
    { role: 'user', content: 'What is the weather in Paris and Tokyo?' },
  ];

  const tools = [{
    type: 'function',
    function: {
      name: 'get_weather',
      parameters: {
        type: 'object',
        properties: { location: { type: 'string' } },
        required: ['location'],
      },
    },
  }];

  // LLM call — may request tool calls
  const response = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages,
      tools,
    });
  });

  // Execute all tool calls — each gets a traced tool span
  const toolMessages = await ctx.executeTools(response, {
    get_weather: async (args) => {
      return await fetchWeather(args.location);
    },
  });

  // Continue conversation with tool results
  messages.push(response.choices[0].message, ...toolMessages);

  const finalResponse = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages,
    });
  });

  return finalResponse.choices[0].message.content;
}, { name: 'weather-agent' });
```

This produces a clean span tree:

```
Trace: weather-agent
├── llm (gpt-4o) — requests tool calls
├── tool (get_weather) — Paris
├── tool (get_weather) — Tokyo
└── llm (gpt-4o) — synthesizes final answer
```

## Agentic Loop with executeTools

For agents that call tools iteratively until done:

```javascript
await foil.trace(async (ctx) => {
  const messages = [{ role: 'user', content: userQuery }];

  while (true) {
    const response = await ctx.llmCall('gpt-4o', async () => {
      return await openai.chat.completions.create({
        model: 'gpt-4o',
        messages,
        tools,
      });
    });

    const choice = response.choices[0];

    // No more tool calls — done
    if (choice.finish_reason !== 'tool_calls') {
      return choice.message.content;
    }

    // Execute tools and continue
    const toolMessages = await ctx.executeTools(response, toolMap);
    messages.push(choice.message, ...toolMessages);
  }
}, { name: 'agent-loop' });
```

## Alternative: wrapOpenAI

For simpler use cases, `wrapOpenAI` automatically traces all OpenAI calls within a trace:

```javascript
const OpenAI = require('openai');
const { Foil } = require('@getfoil/foil-js');

const openai = new OpenAI();
const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
});

await foil.trace(async (ctx) => {
  // Wrap OpenAI client with trace context
  const wrappedOpenAI = foil.wrapOpenAI(openai, { context: ctx });

  // All calls are automatically traced
  const response = await wrappedOpenAI.chat.completions.create({
    model: 'gpt-4o',
    messages: [{ role: 'user', content: 'Hello!' }],
  });

  return response.choices[0].message.content;
});
```

## Alternative: Auto-Instrumentation

For zero-code tracing with no manual wrapping:

```javascript
const OpenAI = require('openai');
const { Foil } = require('@getfoil/foil-js');

const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
  instrumentModules: { openAI: OpenAI },
});

const openai = new OpenAI();

// Automatically traced — no wrapping needed
const response = await openai.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

See [Auto-Instrumentation](/sdks/javascript/auto-instrumentation) for full details.

## What Gets Captured

All tracing methods automatically capture:

| Field | Description |
|-------|-------------|
| Model | The model used (gpt-4o, gpt-4o-mini, etc.) |
| Input | Full message array |
| Output | Assistant response content |
| Tokens | Prompt, completion, and total tokens |
| Latency | Total request duration |
| TTFT | Time to first token (streaming) |
| Tool Calls | Function/tool invocations |
| Errors | Any API errors |

## Streaming Responses

Streaming is fully supported:

```javascript
await foil.trace(async (ctx) => {
  const stream = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: 'Write a haiku' }],
      stream: true,
    });
  });

  for await (const chunk of stream) {
    process.stdout.write(chunk.choices[0]?.delta?.content || '');
  }
});
```

## Multiple Calls in One Trace

Track a complete conversation with multiple LLM calls and tools:

```javascript
await foil.trace(async (ctx) => {
  const messages = [];

  // First call — get plan
  messages.push({ role: 'user', content: 'Plan a trip to Japan' });
  const planResponse = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages,
    });
  });
  messages.push(planResponse.choices[0].message);

  // Tool call — look up flights
  const flights = await ctx.tool('search-flights', async () => {
    return await searchFlights('Japan');
  });

  // Second call — incorporate tool results
  messages.push({ role: 'user', content: `Flights found: ${JSON.stringify(flights)}. Give me details on day 1.` });
  const detailResponse = await ctx.llmCall('gpt-4o', async () => {
    return await openai.chat.completions.create({
      model: 'gpt-4o',
      messages,
    });
  });

  // All calls appear as separate spans in the same trace
  return detailResponse.choices[0].message.content;
}, { name: 'trip-planner' });
```

## Without Trace Context

For simple one-off logging without full tracing:

```javascript
import { Foil } from '@getfoil/foil-js';

const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
});

// Wrap client globally (no trace context)
const wrappedOpenAI = foil.wrapOpenAI(openai);

// Calls logged individually (fire-and-forget)
const response = await wrappedOpenAI.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello' }],
});
```

## Error Handling

Errors are automatically captured in spans:

```javascript
await foil.trace(async (ctx) => {
  try {
    const response = await ctx.llmCall('gpt-4o', async () => {
      return await openai.chat.completions.create({
        model: 'gpt-4o',
        messages: [{ role: 'user', content: 'Hello' }],
      });
    });
    return response;
  } catch (error) {
    // Error automatically recorded in span with:
    // - error.message
    // - error.code (if OpenAI error)
    // - status: 'error'
    throw error;
  }
});
```
