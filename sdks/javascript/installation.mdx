# Installation

The Foil JavaScript SDK provides full-featured tracing, logging, and feedback collection for Node.js applications.

## Requirements

- Node.js 18 or higher
- npm, yarn, or pnpm

## Install via npm

```bash
npm install @getfoil/foil-js
```

Or with yarn:

```bash
yarn add @getfoil/foil-js
```

Or with pnpm:

```bash
pnpm add @getfoil/foil-js
```

## Configuration

### API Key

Get your API key from the [Foil Dashboard](https://app.getfoil.ai) under **Settings > API Keys**.

<Warning>
  Never hardcode API keys in your source code. Use environment variables instead.
</Warning>

```bash
# .env
FOIL_API_KEY=sk_live_xxx_yyy
```

### Initialize Foil

```javascript
import { Foil } from '@getfoil/foil-js';
import OpenAI from 'openai';

const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
  instrumentModules: { openAI: OpenAI },
});
```

### Configuration Options

| Option | Type | Required | Default | Description |
|--------|------|----------|---------|-------------|
| `apiKey` | string | Yes | - | Your Foil API key |
| `agentName` | string | No | `'default-agent'` | Unique identifier for your agent |
| `instrumentModules` | object | No | - | Module map for [auto-instrumentation](/sdks/javascript/auto-instrumentation) (e.g., `{ openAI: OpenAI }`) |
| `defaultModel` | string | No | - | Default model name for spans |
| `debug` | boolean | No | `false` | Enable debug logging |

### Debug Mode

Enable debug mode to see detailed logs of all SDK operations:

```javascript
const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
  debug: true,
});
```

Or set the environment variable:

```bash
FOIL_DEBUG=true
```

Debug output shows:
- Span start/end events with IDs
- Nesting depth visualization
- Timing information
- API call results

## Usage

With auto-instrumentation enabled, your LLM calls are traced automatically — no wrapping needed:

```javascript
import { Foil } from '@getfoil/foil-js';
import OpenAI from 'openai';

const foil = new Foil({
  apiKey: process.env.FOIL_API_KEY,
  agentName: 'my-agent',
  instrumentModules: { openAI: OpenAI },
});

const openai = new OpenAI();

await foil.trace(async (ctx) => {
  // All OpenAI calls are automatically traced — no wrapping needed
  const response = await openai.chat.completions.create({
    model: 'gpt-4o',
    messages: [{ role: 'user', content: 'Hello!' }],
  });

  return response.choices[0].message.content;
}, { name: 'my-trace' });

await foil.shutdown();
```

### Supported Providers

| Provider | Module Key |
|----------|-----------|
| OpenAI | `openAI: OpenAI` |
| Anthropic | `anthropic: Anthropic` |
| Azure OpenAI | `azureOpenAI: AzureOpenAI` |
| Cohere | `cohere: Cohere` |
| Google Generative AI | `google: GoogleGenerativeAI` |
| AWS Bedrock | `bedrock: Bedrock` |

<Note>
  **Using a custom or unsupported model?** Use `ctx.llmCall()` to manually wrap any LLM call in a span. This works with any provider — self-hosted models, local Ollama, or any HTTP API.

  ```javascript
  await foil.trace(async (ctx) => {
    const response = await ctx.llmCall('my-model', async () => {
      return await myCustomLLM.generate({ prompt: 'Hello!' });
    });
  }, { name: 'my-trace' });
  ```

  See the [Quickstart](/sdks/javascript/quickstart) for more details.
</Note>
